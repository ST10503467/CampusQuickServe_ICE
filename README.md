# CampusQuickServe_ICE

This code is for an application which prompts the user to input information about themselves, such as their name and student number, and about what they ordered, such as the price per item and quantity ordered, and then displays a short receipt with that inputed information on it.

By first applying a logical structure to discern whether or not a discount should be applied, the program accepts inputs with JOptionPane boxes and converts them all into strings in order to be displayed on a final JOptionPane box with all of the information on it.
